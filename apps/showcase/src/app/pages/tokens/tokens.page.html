Design Tokens

The sys-* token system — every custom property consumed by @creativo/ui/*. Live values follow the shell's theme/density toggles.

Colors

@for (color of colors; track color) {

--sys-color-{{ color }}

}

Typography

@for (role of textRoles; track role) {

{{ role }} — The quick brown fox jumps

}

Spacing

@for (step of spacing; track step) {

--sys-space-{{ step }}

}

Radii

@for (radius of radii; track radius) {

--control-radius-{{ radius }}

}

Control sizes

@for (size of controlSizes; track size.name) {

--control-size-{{ size.name }} ({{ size.px }}px @ regular density)

}
